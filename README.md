Bengaluru House Price Prediction using Machine Learning

This project was developed to analyze and predict Bengaluru house prices using Machine Learning techniques and data visualization. The model predicts house prices based on various factors such as location, size, number of bedrooms, bathrooms, and total square footage.

Overview

The goal of this project is to build a House Price Prediction System using regression algorithms. It helps in estimating property prices in Bengaluru based on real estate data.

The project includes data cleaning, exploratory data analysis, feature engineering, model training, and prediction.

Features
Data Preprocessing:
Handling missing values
Removing outliers
Feature engineering
Data normalization and cleaning
Exploratory Data Analysis:
House price distribution analysis
Location-wise price comparison
Correlation analysis using heatmaps
Data visualization using graphs and charts
Machine Learning Models:
Linear Regression
Decision Tree Regressor
Random Forest Regressor
Model Evaluation:
R² Score
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
Dataset
Source: Kaggle
Dataset contains Bengaluru housing data including area, location, BHK, bathrooms, and price details.
Technologies Used
Python
Jupyter Notebook
pandas, numpy
matplotlib, seaborn
scikit-learn
Outcome

Successfully developed a Machine Learning model capable of predicting Bengaluru house prices with data-driven insights and visualization techniques.
